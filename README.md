# Getting Started

Поднимается на контроллере

# Первое задание
Надо сделать запрос по адресу: /addresses/ 

Запрос вида:
{
  "addressIds": [
    "1783756",
    "1745891",
    "1790588",
    "1796817"
  ],
  "date": "2015-01-01"
}

# Второе задание

Надо сделать запрос по адресу: /addresses-hierarchy/

Запрос вида:
http://localhost:8080/addresses-hierarchy/?typeName=проезд
